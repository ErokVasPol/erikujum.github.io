Conoce a Erik Vasquez Policarpo

Como apasionado del desarrollo web, mi fascinación por la programación me impulsa a un continuo y emocionante viaje de aprendizaje. Siempre estoy ávido por descubrir nuevas tecnologías y métodos de codificación que mejoren mis habilidades. Mi dedicación se traduce en una constante exploración de innovaciones, buscando perfeccionar mi oficio y crear soluciones vanguardistas para el diseño y desarrollo de sitios web. Mi curiosidad insaciable y mi compromiso con la evolución tecnológica me mantienen en constante actualización, lo que me convierte en un profesional adaptable y proactivo en este dinámico campo del desarrollo web.

Programación

Soy un programador versátil con experiencia en HTML, PHP, Java, C++, y MySQL. Mi habilidad para desarrollar interfaces web sólidas con HTML y PHP se complementa con la capacidad de crear soluciones robustas con Java y C++. Además, domino la gestión de bases de datos utilizando MySQL

PHP C++ Java HTML MySql

Design

Con experiencia en UX, UI, Bootstrap y Materialize, creo experiencias de usuario intuitivas y atractivas. Mi enfoque en UX comprende las necesidades del usuario, mientras que mi habilidad en UI genera interfaces funcionales y atractivas. Con sólido dominio de Bootstrap y Materialize, desarrollo interfaces modernas y responsivas para una experiencia de usuario consistente en todos los dispositivos.

UX Bootstrap Materialize UI

Management

Con experiencia en GitHub, Scrum y MVC, lidero proyectos ágiles con repositorios eficientes, facilito la colaboración en equipos y diseño aplicaciones de manera modular para un mantenimiento sencillo y escalabilidad.
